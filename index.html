AI 활용 의학 논문 작성 워크숍 — 설치 안내

터미널·명령어 타이핑 없습니다. 설치파일 더블클릭 + Code 탭에 자연어 붙여넣기가 전부입니다.

✅ 0번(앱 설치·로그인, Windows는 Git까지)은 워크숍 전날까지 집에서 미리 끝내 오세요. 당일엔 Code 탭에 발화 2개만 붙여넣으면 됩니다.

0사전 준비 — 설치파일 더블클릭만 (터미널 X)

① Claude Desktop 앱 설치 + 로그인 (Mac · Windows 공통)

학교 지원 Claude Pro/Max 계정으로 로그인. 설치파일을 받아 더블클릭만 하면 됩니다.

⬇️Claude Desktop 다운로드claude.ai/download

② Windows 사용자만 — Git for Windows 설치 Windows

설치파일 더블클릭 → 설치 후 Claude 앱을 완전히 종료했다 다시 켜기.

🪟Git for Windowsgit-scm.com/downloads/win — 옵션은 기본값 그대로 Next
💡 왜 이것만 미리? Windows에서는 Code 탭이 처음 켜질 때 Git가 있어야 합니다. 그래서 이건 Code 탭 안에서 깔 수 없고(Code 탭이 그걸 필요로 하니까) 먼저 더블클릭으로 설치합니다. Mac은 보통 이미 있어 건너뜁니다.

③ (권장) Python — 일부 점검 단계(참고문헌·통계)에서 사용

Mac은 보통 이미 설치돼 있습니다. Windows에 없으면 python.org에서 받아 더블클릭 설치 — 설치 첫 화면에서 "Add python.exe to PATH" 체크. (없어도 앞 단계는 진행되고, 해당 점검은 사후로 넘길 수 있습니다.)

①Code 탭 열고 작업 폴더 고르기

Claude 앱 상단 Code 탭 → "폴더 선택(Select folder)" → 오늘 논문용 빈 폴더를 하나 선택(또는 새로 만들기). 이 폴더가 오늘의 작업 공간이 됩니다.

(Windows에서 Code 탭이 처음이라면 0-② Git 설치 후 앱 재시작을 먼저.)

②발화 1 — 셋업 (복사해서 붙여넣기) 가장 중요

아래를 복사해 Code 탭 입력창에 붙여넣으면, Claude가 저장소 복제·skill 복사·도구 설치를 알아서 합니다. 권한을 물으면 승인만 누르세요. (터미널 타이핑 없음)

지금 선택한 이 폴더를 제 논문 작업 폴더로 셋업해줘. 터미널 명령은 네가 실행하고, 권한 물으면 내가 승인할게. 1) https://github.com/samahn0601/sam-workshop 를 clone하고, 그 안 skills/sam-workshop/ 의 18개 skill 폴더(진입점 start-here + 작업 skill 17)와 _shared 폴더를 이 폴더의 .claude/skills/ 바로 아래로 복사해줘. (우산 폴더 sam-workshop을 만들지 말 것 — .claude/skills/start-here 처럼 1단계가 되도록.) 2) git이 없으면 ZIP https://github.com/samahn0601/sam-workshop/archive/refs/heads/main.zip 을 받아서 풀어 같은 위치에 넣어줘. 3) python이 있으면 'pip install --user python-docx' 도 실행해줘 (실패하면 건너뛰고 알려줘). 끝나면 .claude/skills/ 바로 아래 폴더 목록을 보여줘.
⚠️ 우산 폴더 금지. sam-workshop 폴더째 넣으면 2단계가 되어 skill이 전부 안 잡힙니다. 내용물을 .claude/skills/ 바로 아래로. (발화에 이미 들어 있으니 그대로 붙여넣으면 됩니다.)
