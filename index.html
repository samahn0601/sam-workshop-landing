② 발화 1 — 셋업 (복사해서 붙여넣기) 가장 중요

붙여넣으면 Claude가 저장소 복제·skill 복사·도구 설치를 알아서 합니다. 권한을 물으면 승인만 누르세요. (터미널 타이핑 없음)

지금 선택한 이 폴더를 제 논문 작업 폴더로 셋업해줘. 터미널 명령은 네가 실행하고, 권한 물으면 내가 승인할게. 1) https://github.com/samahn0601/sam-workshop 를 clone하고, 그 안 skills/sam-workshop/ 의 18개 skill 폴더(진입점 start-here + 작업 skill 17)와 _shared 폴더를 이 폴더의 .claude/skills/ 바로 아래로 복사해줘. (우산 폴더 sam-workshop을 만들지 말 것 — .claude/skills/start-here 처럼 1단계가 되도록.) 2) git이 없으면 ZIP https://github.com/samahn0601/sam-workshop/archive/refs/heads/main.zip 을 받아서 풀어 같은 위치에 넣어줘. 3) python이 있는지 확인하고, 없으면 관리자 권한 없이(user 설치) 깔아줘 — Windows면 'winget --scope user' 또는 python.org 설치파일을 사용자 모드(PATH 추가)로, Mac이면 이미 있는 python3 사용. 그다음 'pip install --user python-docx'까지. 막히면 건너뛰고 무엇이 막혔는지 알려줘. (python을 방금 깔았으면 PATH 반영에 새 세션이 필요할 수 있어.) 끝나면 .claude/skills/ 바로 아래 폴더 목록을 보여줘.
⚠️ 우산 폴더 금지. sam-workshop 폴더째 넣으면 2단계가 되어 skill이 전부 안 잡힙니다. 내용물을 .claude/skills/ 바로 아래로. (발화에 이미 들어 있으니 그대로 붙여넣으면 됩니다.)
