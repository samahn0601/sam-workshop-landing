· 무과금 연결(OAuth) — Codex OAuth로 연결하면 Code 탭 안에서 Claude가 GPT를 직접 호출합니다. 별도 OPENAI_API_KEY 과금 없이 ChatGPT 구독 쿼터를 씁니다.

연결은 Code 탭에 한국어로 시키면 됩니다 (터미널 명령 몰라도 OK):

ChatGPT 구독으로 GPT를 무과금 연결(OpenAI Codex CLI, OAuth 로그인)하고 싶어. 내 OS에 맞게 Codex를 설치하고 codex login으로 ChatGPT 계정 로그인하는 것까지 안내해줘. 시스템을 바꾸는 명령은 실행 전에 꼭 확인받아.
