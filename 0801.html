Nous Research의 오픈소스 자율 에이전트 — PC에 상주하며 영속 기억 + 예약(cron) 실행으로 ①의 투고 추적을 "매일 아침 자동 점검"으로 상시화. 텔레그램·슬랙에 붙이면 폰에서 비서와 대화.

브레인(모델)·메시징 설정은 외부 계정 발급이 필요해 집에서 천천히 — 단계 전부 🅱 퀵스타트 5번 절에.

⑤ 브라우저·Office 안의 Claude

Claude in Chrome — 크롬 웹스토어에서 "Claude in Chrome" 추가 → 사이드패널에서 웹을 읽고·클릭·폼 입력(승인 후 실행 모드). 크롬 전용.
Claude for Office — PowerPoint(=Word·Excel 동일) 추가 기능에서 "Claude by Anthropic" 검색 → 문서 안에서 Claude. Word는 변경내용 추적으로 제안. ⚠️ 학교·병원 M365는 관리자가 애드인을 막아둘 수 있음 → 안 되면 office.com 웹에서.

⑥ NotebookLM — 문헌 소화 AI · 무료 · 구글 계정

Code가 만드는 AI라면 NotebookLM은 소화시켜주는 AI — 내가 올린 문헌·논문 안에서만 답하고(출처 인용·환각 억제), 요약·오디오/비디오 오버뷰·퀴즈·플래시카드를 자동 생성합니다. 본인 논문 PDF·참고문헌을 넣어 보세요.

notebooklm.google → 개인 구글 계정 로그인(학교 Workspace는 막힌 경우 있음) → 새 노트북 → PDF 업로드 → 출력 언어 한국어. ⭐ 답변의 출처 인용을 클릭해 원문 근거로 점프해 보세요 — "AI가 지어낸 게 아니라 내 소스에서 왔다"를 눈으로(가짜 인용 경계 습관과 직결). ⚠️ 무료 한도: 오디오 ~3/일·비디오는 더 제한적.

⑦ Claude Design + 무과금 OAuth · 더 멀리

Claude Design(claude.ai/design) — 대화로 프로토타입·슬라이드·원페이저를 만들고 Code로 넘겨 실제 구현까지.
구독 OAuth — ChatGPT 구독자는 Codex OAuth로 연결해 API 과금 없이 GPT와 교차 자문·이미지 생성. ※ 상세·복붙 프롬프트는 위 2부 "GPT는 선택" 절 · 비공식 경로.
